# A projektről 

* Egy étteremláncnak a nyilvántartás készítése, mely biztosítja az étterem és
a vendégek számára a zökkenőmentes adminisztrációs munkákat.


# Felhasznált technológiák 

* Angular
* Laravel
* MYSQL szerver
